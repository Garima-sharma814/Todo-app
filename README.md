# Todo app
Todo-app a task tracker. This can take a note of all the tasks we want to do today. We can add tasks, delete tasks from this app.

- Made just to make clear understanding on node, express etc.

## Prerequisite
- Node 
- NPM
- Nodemon (optional)
## Dependencies
- Express
- EJS
- Body-Parser
## Installation
```
npm i 
```
